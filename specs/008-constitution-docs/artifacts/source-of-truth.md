# Source of Truth Policy

## Authority Order

1. Constitution (`.specify/memory/constitution.md`)
2. Feature spec and contracts (`specs/008-constitution-docs/spec.md`, `specs/008-constitution-docs/contracts/`)
3. Coverage matrix (`docs/constitution/coverage-matrix.md`)
4. Delivery/readability surfaces (`README.md`, directory READMEs, evidence docs)

## README Synchronization Rules

- README files are synchronized delivery surfaces and must not be treated as normative authority.
- When conflicts are found, update README/evidence documents to match higher-order sources.
- Root `README.md` remains the canonical runbook entrypoint for reviewer workflows.
